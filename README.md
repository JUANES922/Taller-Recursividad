DESCRIPCION DEL TALLER 

Ejercicio 1. Factorial  calcula el factorial de un numero entero usando recursividad.
complejidad: O(n)
Ejercicio 2. Sumatoria  calcula la sumatoria de 1 hasta n de forma iterativa.
complejidad: O(n)
Ejercicio 3. Sumatoria fracciones calcula la serie 1 + 1/2 + 1/3 + … + 1/n.
complejidad: O(n)
Ejercicio 4. Invertir numero invierte los digitos de un numero por ejemplo: 123 , 321.
complejidad: O(n)
Ejercicio 5. Suma de digitos suma los digitos de un numero usando aritmetica por ejemplo: 123 , 6.
complejidad: O(n)
Ejercicio 6. Suma de digitos suma los digitos de un numero usando conversión un String.
complejidad: O(1)
Ejercicio 7. Potencia calcula base exponente usando recursividad.
complejidad: O(n)
Ejercicio 8. MCD  calcula el maximo comun divisor de dos numeros con el algoritmo de Euclides recursivo.
complejidad: O(log n)
Ejercicio 9. Cociente calcula el cociente de una division entera mediante restas sucesivas.
complejidad: O(n)
Ejercicio 10. Multiplicacion multiplica dos numeros mediante sumas sucesivas.
complejidad: O(n)
Ejercicio 11. Suma arreglo  genera un arreglo de n enteros aleatorios y calcula la suma de sus elementos.
complejidad: O(n)
Ejercicio 12. Suma matriz  genera una matriz de m*n enteros aleatorios y calcula la suma de todos sus elementos.
complejidad: O(2n)
Ejercicio 13. Fibonacci imprime la serie de Fibonacci hasta un limite dado usando recursividad.
complejidad: O(n)
Ejercicio 14. Ackermann  calcula la funcion de Ackermann(m, n) usando recursividad.
complejidad: O(2n)

el programa se debe ejecutar en netbeans se debe tener un jdk 18 o mayor y cuando se tenga los requerimientos los ejercicios estan en distintas,
clases en total 14, los ejercicios se ejecutan o corren en una clase main la cual ejecuta todos los ejercios y muestra sus resultados.
